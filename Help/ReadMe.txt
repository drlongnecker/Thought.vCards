
Overview

  This folder contains compiled documentation as well as a Sandcastle Help File Builder
  project that can be used to rebuild the documentation.  You should have some experience
  with Sandcastle and SHFB.

Required Tools

  1. Download and install the Sandcastle MSI from http://sandcastle.codeplex.com/
  2. Download and install the Sandcastle Help File Builder from http://www.codeplex.com/SHFB/
  3. Download and install HTML Help Workshop from http://msdn.microsoft.com/en-us/library/ms669985(VS.85).aspx
